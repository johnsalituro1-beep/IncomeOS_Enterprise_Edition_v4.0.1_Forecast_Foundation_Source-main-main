# Enterprise Edition v1.2 Release Notes

IncomeOS v1.2 adds the Live ETF Data Platform foundation. The release includes provider adapters, an executable sandbox sync, record validation, duplicate detection, evidence lineage, ingestion audit records, coverage and freshness monitoring, a Data Operations dashboard, and a Supabase migration.

The bundled sandbox data is modeled. A licensed provider and production credentials are still required for real market data.
