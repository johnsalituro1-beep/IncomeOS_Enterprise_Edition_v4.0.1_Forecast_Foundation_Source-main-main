# Version 4.0 — Milestone 2 Preview

## Delivered
- Final approved gold logo integrated into the terminal sidebar and favicon.
- Income Command Center redesigned to match the frozen Excel visual specification.
- Functional browser-persistent holdings engine.
- Add and remove holdings.
- Live portfolio value, cost basis, gain/loss, yield and income calculations.
- Editable weekly income goal.
- Income analytics and per-holding income contribution.
- Responsive desktop/tablet/mobile presentation.

## Data status
Starter holdings are demonstration data. Changes are stored locally in the browser during this preview. The next sprint connects holdings CRUD to Supabase.

## Deployment
Upload all files to the repository root and deploy without cache. Do not upload an old pnpm-lock.yaml.
