# Program B — Income Digital Twin 2.0

This increment introduces a deterministic projection engine, scenario cloning architecture, annual cash-flow ledger, retirement-age logic, inflation-adjusted values, taxation, partial reinvestment, contributions, withdrawals, recurring life events, early sequence shocks, and seeded Monte Carlo ranges.

## Production boundaries
The offline release uses modeled development assumptions. It does not supply tax, legal, or investment advice. Before production use, connect validated ETF data, calibrate capital-market assumptions, add scenario persistence, expand tests, and complete regulatory review.
