# Program A — Phase 1 Complete

Version: `17.0.0-program-a-phase-1-complete`

## Delivered

This release turns the Version 16 ETF master-data foundation into an executable ETF intelligence layer.

- ETF Knowledge Graph 2.0 relationship framework
- Universal Holdings Engine with latest-snapshot and normalized-exposure utilities
- Distribution Intelligence with consistency, growth, cadence and payment statistics
- Eight-component explainable ETF scoring engine
- Holdings, sector, country and strategy overlap analytics
- Issuer and benchmark profile registries
- Research metadata and provider-confidence gating
- Phase 1 control-room UI with scorecards and pair diagnostics
- Database migration for scorecards, issuer profiles, benchmark profiles, normalized exposures and research metadata

## Data boundary

All included ETF records and analytical outputs are development fixtures. They are not verified market facts and must not be used for investment decisions. Production deployment requires licensed or authoritative data sources, historical backfills, provider contracts, validation controls and legal review.

## Next build

Program A Phase 2 should add the historical data warehouse: adjusted price and NAV series, distribution revisions, holdings versioning, corporate actions, data lineage, scheduled ingestion and change detection.
