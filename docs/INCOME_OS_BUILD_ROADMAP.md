# Income OS Build Roadmap

## Product direction
ETF Dividend Calendar Pro is being developed as an Income Operating System. The Portfolio Command Center is the main workspace and the Income Intelligence Engine is the central analytical service.

## Initiatives 1–10
1. Central Income Intelligence Engine — active; rule-based engine and shared calculations implemented.
2. Income OS information architecture — active; Income, Growth, Intelligence and Research workspaces introduced.
3. Portfolio Command Center v2 — active; health analysis, briefing, timeline and OS navigation integrated.
4. Intelligence Engine v3 — planned after live data and verified historical feeds are available.
5. Guided onboarding — interface and persistence model next.
6. Proprietary scoring system — first Portfolio Health Score implementation complete; sub-scores are explainable.
7. Institutional reporting — report contracts and export layouts pending.
8. Mobile companion — responsive command-center foundation underway.
9. Portfolio-aware assistant — conversational contract pending; deterministic engine remains source of truth.
10. Versioned roadmap — release gates retained across Tracks A–C.

## New foundation in this build
- Portfolio Health Score with weighted dimensions.
- Income OS workspace navigation.
- Dedicated Intelligence Center route.
- Portfolio Flight Recorder with one daily local snapshot and a 365-day retention cap.
- ETF knowledge-base domain model and initial demo profiles.

## Release status
This branch is an offline development branch. It is not production-approved until dependency installation, TypeScript validation, the Vite production build, deployment and smoke testing pass.
