# Income Intelligence Engine™

The first version is deterministic and explainable rather than generative. It converts portfolio metrics into concise observations using documented thresholds.

Current signals:
- weekly goal progress and income gap
- largest income producer
- largest holding concentration
- covered-call strategy concentration
- elevated forward-yield warning

Design principles:
- every message must be traceable to portfolio data
- no unsupported predictions
- distinguish information from advice
- show the metric behind warnings
- link users to the relevant workspace for action

Future versions may add distribution trend changes, payment gaps, NAV behavior, scenario comparisons, and provider-backed announcement monitoring.
