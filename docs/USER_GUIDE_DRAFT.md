# User Guide — Draft

## Daily workflow
Open Income Command Center, read the Daily Briefing, review the Income Timeline, then inspect any item marked Review or Action Required.

## Adding a holding
Open Portfolio Command Center, select Add Holding, enter ticker, shares, cost, price, annual distribution per share, frequency, and category.

## Changing the income goal
Open Settings or Income Analytics and update the weekly income target. Goal progress updates throughout the platform.

## Modeling a scenario
Open Strategy Lab, enter additional capital, target yield, and reinvestment percentage. Review projected weekly and annual income before saving the scenario.
